# 사자성어 퀴즈

60초 안에 사자성어 문제를 빠르게 푸는 React 기반 싱글 페이지 퀴즈 게임입니다.  
빈칸 한자 맞히기와 뜻풀이 보고 사자성어 맞히기 문제를 섞어 출제하며, 짧은 제한 시간과 즉각적인 피드백으로 경쾌한 게임감을 목표로 했습니다.

## 주요 기능

- 60초 전체 게임 타이머
- 난이도별 문제 타이머
  - 쉬움: 15초
  - 보통: 10초
  - 어려움: 7초
- 문제 유형 2가지
  - 빈칸 한자 선택
  - 뜻풀이 보고 사자성어 선택
- 선택지는 한자만 표시
- 같은 판 안에서 동일 사자성어 중복 출제 방지
- 정답, 오답, 시간 초과 피드백
- 결과 화면에서 정답 수, 정확도, 문제 복기 제공
- 결과 문제 복기를 텍스트 파일 또는 이미지 파일로 저장
- 최고 기록, 최고 정확도, 플레이 횟수 로컬 저장
- 오늘의 리더보드 제공
  - 브라우저 로컬 기준으로 상위 10개 기록 저장
  - 날짜가 바뀌면 자동 초기화
- `1`, `2`, `3`, `4` 키보드 선택 지원
- 데스크톱과 모바일 반응형 UI

## 기술 스택

- React
- Vite
- CSS
- React Icons
- LocalStorage
- 로컬 JSON 데이터셋

## 리더보드

리더보드는 서버 없이 브라우저 `localStorage`에 저장됩니다.  
저장된 날짜가 오늘 날짜와 다르면 기록을 비우고 새 일간 리더보드를 시작합니다.

정렬 기준은 아래 순서입니다.

1. 정답 수 높은 순
2. 정확도 높은 순
3. 총 풀이 시간 짧은 순
4. 최근 플레이 순

## 결과 내보내기

결과 화면에서 문제로 나온 사자성어, 뜻, 문제 유형, 내 답, 정답, 정답 여부, 소요 시간을 따로 저장할 수 있습니다.

- 텍스트 저장: `.txt` 파일 다운로드
- 이미지 저장: `.png` 파일 다운로드

## 실행 방법

```bash
npm install
npm run dev
```

개발 서버가 실행되면 브라우저에서 아래 주소를 엽니다.

```text
http://127.0.0.1:5173/
```

## 빌드

```bash
npm run build
```

빌드 결과물은 `dist/`에 생성됩니다.

## 데이터셋

사자성어 데이터는 `src/data/idioms.json`에 있습니다.  
현재 100개의 사자성어가 포함되어 있으며, 각 항목은 아래 형식을 따릅니다.

```json
{
  "id": "001",
  "idiom": "一石二鳥",
  "reading": "일석이조",
  "meaning": "한 가지 행동으로 두 가지 이득을 얻는다는 뜻.",
  "category": "지혜"
}
```

게임 로직은 이 데이터를 기반으로 자연스러운 오답과 랜덤 오답을 섞어 선택지를 구성합니다.
